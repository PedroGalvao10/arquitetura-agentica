# Agentic OS: A next-gen 3-layer autonomous architecture

Bridging the gap between LLM probabilistic reasoning and deterministic implementation via Directives, Orchestration, and Layered Execution. Optimized for High-End UI and Evolutionary Code.

## 🏗️ The 3-Layer Architecture

This project follows a strict separation of concerns to maximize reliability and scalability:

1.  **Layer 1: Directive (The Brain)**
    *   Markdown-based Standard Operating Procedures (SOPs).
    *   Located in `directives/`.
    *   Defines goals, inputs, tools, and edge cases in natural language.

2.  **Layer 2: Orchestrator (The Decision Maker)**
    *   Intelligent routing and state management.
    *   Handled by AI agents (Antigravity/OpenClaude).
    *   Bridges the gap between Intent and Execution scripts.

3.  **Layer 3: Execution (The Action)**
    *   Deterministic Python and PowerShell scripts.
    *   Located in `execution/` and `projects/`.
    *   Reliable, testable, and high-performance implementation.

## 🚀 Key Features

*   **Evolutionary Programming (ADAS)**: Self-optimizing code and directives.
*   **Premium UI/UX**: Apple-inspired immersive experiences using React 19, Tailwind v4, and Three.js.
*   **Autonomous Sync**: Automatic mirroring between local environment, backups, and GitHub.
*   **Human-First Narrative**: Technically dense but linguistically fluid documentation (Zero Dash Policy).

## 🛠️ Tech Stack

*   **Frontend**: React 19, Vite, TypeScript, Tailwind CSS v4, Framer Motion, GSAP.
*   **Backend/Agents**: Ollama (Local LLMs), NVIDIA NIM (Cloud LLMs).
*   **Orchestration**: Python 3.11+, PowerShell Core.
